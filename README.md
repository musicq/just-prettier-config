# just-prettier-config

A preset prettier config of `just` projects.

```javascript
module.exports = {
  tabWidth: 2,
  semi: false,
  printWidth: 80,
  singleQuote: true,
  quoteProps: 'as-needed',
  jsxSingleQuote: false,
  jsxBracketSameLine: false,
  arrowParens: 'avoid',
  bracketSpacing: false,
  trailingComma: 'all',
  endOfLine: 'lf',
}
```
